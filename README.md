# Ticket Booking System Backend

A Java-based Ticket Booking System that allows users to search trains, book tickets, and manage bookings.

## Features

* User registration
* User login
* Train management
* Ticket booking
* Ticket cancellation
* Local JSON-based data storage

## Technologies Used

* Java
* Gradle
* Jackson (JSON Processing)

## Project Structure

src/main/java
├── Entities
├── Service
└── localdb

## Future Improvements

* Spring Boot Migration
* REST APIs
* PostgreSQL Database
* JWT Authentication
* Admin Panel

## How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run App.java
